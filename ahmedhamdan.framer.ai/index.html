Data Access
Facilitator (DAF)

This project simplifies data recovery for front-end developers and QA team by providing a dedicated backend system. Using Express-based microservices, it ensures easy API access for creating and managing mock stores/merchants with varying restrictions. Each store/merchant is accessible through unique IDs, facilitating essential design and testing tasks. Additionally, it allows data addition to the database for specific test cases.
